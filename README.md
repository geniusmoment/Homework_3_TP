## Система для анализа данных в докере

проект реализован в рамках домашнего задания, датасет генерируется автоматически. На выходе получаем html-файл с аналитикой минимальных, максимальных и средних значений

## Список достпуных команд:

ㅤㅤbuild_generator - собрирает образ для контейнера генератора

ㅤㅤrun_generator - запускает контейнер, который сгенерирует data/data.csv локально

ㅤㅤcreate_local_data - в директории local_data создает data.csv (для локальной отладки)

ㅤㅤbuild_reporter - собирает образ для контейнера аналитика

ㅤㅤrun_reporter - запускает контейнер, который сгенерирует html отчет локально в директории data

ㅤㅤstructure - Выводит структуру всех файлов и директорий проекта начиная с текущей папки

ㅤㅤclear_data - Удаляет все сгенерированные данные — файлы `.csv` и `.html` из папки `data/`. После выполнения этой команды папка должна остаться пустой, а повторный запуск `run_generator` и `run_reporter` должен всё пересоздать.

ㅤㅤinside_generator - Запускает контейнер генератора и выводит её содержимое data изнутри контейнера. Так можно убедиться, что контейнер видит данные на хосте.

ㅤㅤinside_reporter - Аналогично предыдущей команде, но для контейнера аналитика. Проверяет, что контейнер reporter видит файлы data с хоста

## Установка

Вот полный алгоритм действий (рабочий):

```
git clone https://github.com/geniusmoment/Homework_3_TP.git
cd Homework_3_TP
./run.sh build_generator
./run.sh run_generator
./run.sh build_reporter
./run.sh run_reporter
```
